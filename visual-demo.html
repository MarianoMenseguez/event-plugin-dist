Divi Module

Add "ECP Events" module in Divi Builder
Configure settings:
- Layout: Grid
- Events per page: 6
- Show filters: Yes
- Show search: Yes

🚀 Quick Setup

  1. Upload plugin files to /wp-content/plugins/event-calendar-plugin/
  2. Activate the plugin in WordPress Admin → Plugins
  3. Go to Event Calendar → Demo Setup
  4. Click "Create Demo Data" to generate sample events
  5. Create a new page and add the Divi Events module
  6. Configure module settings and preview your events page
